## API Request/Response Flow

### Example: Calculate Emissions

**Request:**
```typescript
POST /api/trpc/calculate.create

{
  "origin": "New York, USA",
  "destination": "Los Angeles, USA",
  "distance": 4500,
  "weight": 10,
  "transportMode": "TRUCK",
  "fuelType": "DIESEL",
  "weatherFactor": 0,
  "loadFactor": 80,
  "trafficFactor": 0.15
}
```

**Processing:**
1. Validate input with Zod schema
2. Authenticate user (check JWT)
3. Fetch emission factor from database
4. Calculate base emissions: `4500 km Ã— 10 tonnes Ã— 0.097 kg/tonne-km = 4,365 kg`
5. Apply adjustments: `4,365 Ã— (1 + 0 + 0.2 + 0.15) = 5,893 kg`
6. Save to database
7. Create audit log entry
8. Update goal progress (if applicable)
